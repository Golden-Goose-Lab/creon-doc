# types

#### Candle

_**class Candle\(TypedDict\)**_  
캔들 단위 타입입니다.  
  
_속성_  
**start\_time \(타입 datetime\)**  
캔들의 시작 시간입니다.  
  
**end\_time \(타입 datetime\)**  
캔들의 종료 시간입니다.  
  
**open\_price \(타입 int\)**  
캔들의 시가입니다.  
  
**high\_price \(타입 int\)**  
캔들의 고가입니다.  
  
**low\_price \(타입 int\)**  
캔들의 저가입니다.  
  
**close\_price \(타입 int\)**  
캔들의 종가입니다.  
  
**volume \(타입 int\)**  
캔들의 거래량입니다.
